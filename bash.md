## ls ##


```
//Show all files and show full file info
ls -aF
// Order by time of change desc
ls -t
//Order by name
ls -x
// Shows as a table
ls -l
```

## truncate ##

truncate file to a zero size:

```
truncate -s 0 p.txt
```
